### Project Hail Mary
Upcoming sci-fi movie based on the novel by Andy Weir (author of *The Martian*). A lone astronaut must save Earth from an extinction-level threat with only a few clues and an unexpected companion.

### The Human Interface — Jef Raskin
A foundational book on user interface design by Jef Raskin, creator of the Macintosh project at Apple. Covers cognetics, modelessness, and principles for humane interface design.
- [Goodreads – The Humane Interface](https://www.goodreads.com/book/show/344726.The_Humane_Interface)
- [Wikipedia – Jef Raskin](https://en.wikipedia.org/wiki/Jef_Raskin)

### FreshRSS
A free, self-hosted RSS feed aggregator. Lightweight and easy to deploy, great for keeping up with blogs, news, and podcasts in one place.
- [FreshRSS Official Site](https://freshrss.org/index.html)
- [GitHub – FreshRSS](https://github.com/FreshRSS/FreshRSS)

## IT & Virtualization

### VMware Licensing
VMware's licensing model has undergone significant changes (especially post-Broadcom acquisition), moving toward subscription-based and bundled offerings. Worth tracking if you run VMware infrastructure.
- [VMware by Broadcom – Licensing](https://www.vmware.com/products/desktop-hypervisor/workstation-and-fusion)
- [VMware Licensing FAQ](https://knowledge.broadcom.com/external/article?legacyId=2107518)

### GPU Virtualization
Technologies like NVIDIA vGPU, Intel GVT-g, and SR-IOV that allow a physical GPU to be shared across multiple virtual machines — critical for VDI, AI workloads, and remote graphics.
- [NVIDIA vGPU](https://www.nvidia.com/en-us/data-center/virtual-solutions/)
- [Wikipedia – GPU Virtualization](https://en.wikipedia.org/wiki/GPU_virtualization)
